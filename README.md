# Dian_Algorithm_test
Dian团队2025春招算法测试题

## 环境配置

创建conda Python 3.11.11虚拟环境

```bash
conda create -n dian python=3.11.11
```

激活虚拟环境

```bash
conda activate dian
```

安装依赖

```bash
pip install -r requirements.txt
```
